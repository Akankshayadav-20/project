how to push local repository on github.

Create Project Folder use command mkdir "foldername"

go in project using cd project command

then use git init command for initial push

add readme.md file then check status of your project

Add all file which you are created in this project using git add . command

commit all using git commit -m "Meaningfull Message"

use git remote add origin "link" command. to connect local repo to remote(github)

git remote -v command . to verify remote

git branch to check a branch

use git push origin main command to push your code on github
